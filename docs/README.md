# ProvChain Public Documentation

This directory contains the public-facing documentation for ProvChain.

The documentation included here is limited to material that supports:

- system architecture and ontology-package design
- benchmark methodology and exported artifacts
- deployment and configuration examples
- user and developer onboarding

Excluded from the public repository:

- manuscript working files
- internal review notes
- thesis material
- historical planning archives
- generated documentation build artifacts

## Recommended Reading Order

1. [../README.md](../README.md)
2. [INDEX.md](INDEX.md)
3. [architecture/README.md](architecture/README.md)
4. [benchmarking/README.md](benchmarking/README.md)

## Key Public Sections

- `architecture/` - shared-ontology network model, ADRs, and architecture figures
- `benchmarking/` - reproducible benchmark artifacts, summary tables, and figure generation inputs
- `deployment/` - setup and deployment guidance
- `api/` - API documentation
- `user-guide/` and `user-manual/` - usage guidance
- `ontologies/` - ontology assets and examples

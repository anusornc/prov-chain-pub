# ProvChain Public Documentation Index

This index lists the documentation that is intentionally included in the public repository.

The public documentation focuses on:

- architecture and ontology-package design
- benchmark methodology and exported results
- deployment and configuration examples
- user and developer guides

Internal review notes, manuscript working files, thesis material, and planning archives are intentionally excluded.
